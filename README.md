# 🌿 Plant Disease Detection

## 📌 Overview
This project focuses on detecting plant diseases using deep learning techniques.

## 🛠 Tools & Technologies
- Python
- TensorFlow / Keras
- OpenCV

## 🔍 Key Steps
- Image data preprocessing
- Model building using CNN
- Training and validation
- Performance evaluation

## 📊 Results
- Achieved good accuracy in classifying plant diseases
- Improved predictions through tuning

## 🚀 Outcome
Built a deep learning model to identify plant diseases from leaf images, helping in early detection.
